Analysing and modelling epidemic data Lusaka workshop 2022

Welcome to the Imperial College London / Zambart workshop "Analysing and modelling epidemic data" (infectiousdiseasemodels.org).

This course will be taught using an interface built using an R package called shiny using the R programming language. Over the 10 day programme you will be using this interface to explore and understand the basics of analysing epidemic data and dynamic transmission modelling.

The links below will take you to the practical and project materials and interface you will use over the next two weeks. Links to course materials and solutions will become active as the course progresses so be sure to check back regularly.
